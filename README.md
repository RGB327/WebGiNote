# WebGiNote
웹기 노트 적는곳

# 1일차
* **!DOCTYPE html** - 이 문서가 html 문서임을 알리는 태그 
* **title** - 사이트 타이틀 쓰는 태그
* **head** - 대충 ~~대가리~~머리임
* **p** - 단락으로 묶어줌
* **div** - 블록으로 묶어줌(노션 블록 비스무리)
* **h1~6** - 제목글 생성
    - **title** - 툴팁 생성
* **br** - 줄넘김
* **hr** - 가로선 생성
* **pre** - 개발자가 쓴 그대로 출력(줄넘김 등)
-----
# 2일차
* **ol** - 순서 있는 리스트 (1 - 2 - 3 / a - b - c)
* **ul** - 순서 없는 리스트
* **li** - 리스트 안에 넣는 하나의 내용
## 리스트는 중첩 가능함

* **table** - 표 테이블 생성
* **caption** - 표 제목
* **tr** - 표 한 행 생성
* **thead** - 테이블 머리 생성
    - **th** - 머리 텍스트 생성(보통 머리와 발에서 사용)
* **tbody** - 표 몸통 생성
    - **td** - 테이블 몸통 텍스트 생성(보통 몸통에서 사용)
* **tfoot** - 테이블 발 생성

-----
# 3일차
* a href = "대충 링크" **target "_parent" 등..** - 링크를 눌렀을 때 실행되는 위치 설정
* h3 **id = "대충 이름"** - href로 해당 id를 입력했을 때 해당 id의 위치로 문서 이동(예시: a href= "#대충 이름")
* **href = "이미지" download** - 해당 사진을 다운로드 (크롬 보안문제로 새 탭에서 열림)
* **iframe** - 다른 html 문서를 담을 수 있는 창을 생성
    - **srcdoc** - 다른 html 문서를 가져오는 대신 html을 해당 파일에 직접 짜서 보여줌(src보다 우선)
